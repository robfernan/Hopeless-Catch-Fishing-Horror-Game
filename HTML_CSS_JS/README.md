# Hopeless Catch - Peaceful Fishing Horror Game

A cozy pixel art fishing game with a subtle secret. Unwind by the lake, catch 9 unique fish species, and discover what lurks in the tranquil waters after dark.

## Quick Start

1. Open `index.html` in a web browser
2. Press SPACE to start fishing
3. Press B to select bait (worms are default)
4. Press TAB to view your journal
5. Press H to rest in the cabin and advance time
6. Press ESC to pause/menu

## Controls

| Key | Action |
|-----|--------|
| WASD / Arrows | Move |
| SPACE | Cast/Reel/Hook |
| B | Open Tackle Box |
| TAB | Open Journal |
| H | Rest in Cabin |
| ESC | Pause/Menu |

## Features

### Day Experience
- Beautiful sky transitions (sunrise, midday, sunset)
- Animated water with ripples
- Peaceful music (C major scale)
- 5 day fish species
- Relaxing atmosphere

### Night Experience
- Darkened visuals with moonlight
- Eerie music (A minor scale)
- 4 night creatures
- Optional anomalies
- Mysterious atmosphere

### Gameplay
- 7-phase fishing system
- Dynamic day/night cycle
- Weather effects
- Strategic bait selection
- Fish encyclopedia
- Catch history tracking

### Peaceful Mode
- Toggle in settings
- Disables night creatures
- No anomalies
- Stress-free fishing

## Fish Species

### Day Fish
- Sunfish (common)
- Bass (common)

- Golden Carp (rare)

### Night Fish
- Pale Crawler (uncommon)
- Bleeding Carp (uncommon)
- Whispering Eel (rare)
- Fishman (very rare)

## Game Systems

### Fishing Mechanics
1. Press SPACE to get ready
2. Press SPACE to start casting (power meter builds)
3. Press SPACE to cast (bobber appears)
4. Wait for bite (bobber flashes red)
5. Press SPACE to hook
6. Press SPACE to reel in
7. Catch window appears

### Day/Night Cycle
- Time progresses automatically
- Sky and water colors change
- Music switches automatically
- Fish availability changes
- Anomalies increase at night

### Bait System
- Worms (default, balanced)
- Minnows (attracts faster bites)
- Cheese (slower bites)
- Corn (balanced)

### Journal
- Track total catches
- View fish encyclopedia
- See catch history by day
- Read fish lore
- Monitor statistics

## Settings

- **Peaceful Mode**: Disable night creatures and anomalies
- **Volume**: Adjust master, music, SFX, and ambient volumes
- **Display**: Fixed 800x600 pixel art resolution

## Technical

- **Engine**: HTML5 Canvas + Web Audio API
- **Resolution**: 800x600 pixels
- **FPS**: 60 (locked)
- **Audio**: Procedurally generated (no audio files needed)
- **Save**: Browser localStorage
